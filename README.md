# 📺 YouTube 右方向键长按倍速播放

这是一个 Tampermonkey (油猴) 脚本，重构了 YouTube 网页版 **右方向键 (`→`)** 的交互逻辑，改为：

- **短按 (< 0.25s)**：快进 **5 秒**。
- **长按 (> 0.25s)**：平滑进入 **3.0 倍速**（默认），松手即停。

暂停状态下，不管 **短按** 和 **长按** 都会进入对应播放状态。改成了与 **B站** 相同的操作习惯。



### 视觉反馈

- **经典 UI**：采用半透明黑色胶囊风格，沉浸感强，支持全屏显示。
  - 短按提示：`+ 5s`
  - 长按提示：`>>> 倍速播放中 (3x)`

### ⚙️ 个性化配置 (Configuration)

脚本头部内置了 `CONFIG` 对象，你可以根据喜好轻松修改参数：

```javascript
const CONFIG = {
    // 长按倍速：默认 3.0 (B站习惯)。觉得太快可改回 2.0
    LONG_PRESS_SPEED: 3.0,

    // 短按快进：默认 5秒
    SEEK_SECONDS: 5,

    // 长按灵敏度：默认 250ms。数值越小，触发加速越快
    PRESS_THRESHOLD: 250,

    // UI 位置：距离顶部的距离 (支持 % 或 px)
    TIP_TOP_POSITION: '5%',

    // UI 颜色：默认黑色半透明
    COLOR_UI: 'rgba(0, 0, 0, 0.7)'
};
```

## 🛠️ 安装指南

1. 安装浏览器扩展 **Tampermonkey** (Chrome / Edge / Firefox / Safari)。
2. 点击扩展图标 -> **添加新脚本**。
3. 删除编辑器内所有默认代码。
4. 将脚本代码完整粘贴进去。
5. 按 `Ctrl + S` 保存即可生效。

## ❓ 常见问题 (FAQ)

**Q: 提示框挡住字幕了怎么办？**

A: 默认位置在顶部 8%，通常不会挡字幕。如果你的屏幕比例特殊，可以在配置里把 `TIP_TOP_POSITION` 改得更小（如 `2%`）。
